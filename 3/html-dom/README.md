# Лаба 6: HTML Basics, DOM API

## Что реализовано

- Базовая HTML-страница с заголовками, текстом, формулой и блоком цен.
- DOM-защита страницы: блокировка выделения, копирования, контекстного меню и части сочетаний клавиш.
- Калькулятор из 16 кнопок, работающий прямо в браузере.

## По частям

1. Разметка:
- `h1`, `h2`, `p`, `sup`, блочный контейнер с ценами.
- Есть пример со старой и новой ценой.

2. DOM API:
- Через `addEventListener(...)` обрабатываются `contextmenu`, `copy`, `cut`, `dragstart`, `selectstart`, `keydown`.
- Статус защиты и журнал событий обновляются динамически.

3. Калькулятор:
- Кнопки `0-9`, `+`, `-`, `*`, `/`, `C`, `=`.
- Хранится текущее число, предыдущее число и выбранная операция.

## Примечание

- PDF удалось восстановить только частично, поэтому реализация собрана по явно читаемым фрагментам задания: базовая HTML-страница, защита страницы через DOM API и калькулятор.
