# Лабораторные `frontend/3`

## Что лежит в папке

- `html-dom/` — работа по теме `HTML Basics, DOM API`.
- `css/` — работа по теме `CSS`.

## Принцип оформления

- Структура сделана в том же духе, что и прошлые лабы: отдельные папки, чистый HTML/CSS/JS без сборки.
- В `html-dom` собраны ключевые части из задания: базовая разметка, работа с DOM API и калькулятор.
- В `css` сделана отдельная оформленная страница-витрина на чистом CSS.

## Как открыть

- Открой `html-dom/index.html`.
- Открой `css/index.html`.
