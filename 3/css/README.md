# Лаба 7: CSS

## Что реализовано

- Отдельная страница-витрина на чистом HTML/CSS.
- Используются CSS-переменные, `grid`, `flex`, hover-состояния, псевдоэлементы и media queries.
- Есть оформленные карточки товаров, hero-блок, формула как декоративный блок и адаптивная форма подписки.

## Что можно посмотреть в коде

- `:root` — общая палитра, радиусы и тени.
- `.hero-grid`, `.catalog-grid`, `.feature-grid` — сетки на `grid`.
- `.brand::after`, `.product-card::after` — примеры псевдоэлементов.
- `@media (max-width: ...)` — адаптация под планшет и мобильный экран.

## Идея страницы

- Визуально это продолжение темы из HTML-лабы: учебная витрина магазина, но уже с полноценным CSS-оформлением.
